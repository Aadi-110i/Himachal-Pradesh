# 🏔️ Himachal Treasures

A digital sanctuary dedicated to preserving and showcasing the rich cultural heritage of Himachal Pradesh.

## 🌟 Overview
**Himachal Treasures** is an immersive web experience that brings the spiritual and architectural wonders of the Himalayas to your screen. From 360° virtual tours of sacred monasteries to digital archives of ancient texts, the platform serves as a bridge between tradition and technology.

## ✨ Key Features
- **🔭 360° Virtual Tours:** Explore the intricate interiors and serene courtyards of monasteries like Rumtek.
- **📜 Digital Archives:** Access a curated collection of Buddhist murals, manuscripts, and relics.
- **📅 Cultural Calendar:** Stay updated on monastic festivals and spiritual events.
- **🗺️ Interactive Discovery:** Engage with the heritage through guided tours and knowledge quests.
- **🎵 Audio Sanctuary:** Listen to traditional chants and ceremonial music.

## 🛠️ Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **3D Engine:** React Three Fiber + Three.js
- **Icons:** Lucide React

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Aadi-110i/Himachal-Pradesh.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
